# bananas

monkeytype in your terminal

**for linux environments**

## Installation

### Build from scratch

1. Clone repository

```bash
https://github.com/hungrymonkeystudio/bananas.git
```

2. Build project with Go version 1.24

```bash
go build .
```

3. Run make install

```bash
make install
```

4. Run make uninstall to remove program

```bash
make uninstall
```
