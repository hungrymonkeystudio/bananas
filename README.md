# bananas

monkeytype in your terminal

**for linux environments**

## Install 

### Install Script

```bash
git clone https://github.com/hungrymonkeystudio/bananas
cd scripts
chmod +x install.sh
./install.sh
```

## Uninstall

### Uninstall Script

```bash
git clone https://github.com/hungrymonkeystudio/bananas
cd scripts
chmod +x uninstall.sh
./uninstall.sh
```
